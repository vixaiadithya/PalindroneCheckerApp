# PalindromeCheckerApp

## Project Objective

The objective of this project is to develop a Java application that checks whether a given word or string is a **palindrome** or not.
A palindrome is a word or number that reads the same forward and backward (example: **madam, level, radar**).
This project also demonstrates the use of **Git, GitHub, branching, committing, and pull requests** while developing a Java application.

---

## Procedure

1. Create a **GitHub repository** named `PalindromeCheckerApp`.
2. Enable the **README.md file** while creating the repository.
3. Copy the repository link and **clone the repository** to the local system.
4. Open the project in **IntelliJ IDEA**.
5. Create a **new module named App**.
6. Inside the `src` folder create a **Java class named `PalindromeCheckerApp`**.
7. Write the Java code to implement the **palindrome checking logic**.
8. Create a **new branch for each use case** using Git.
9. Add the files using `git add`.
10. Commit the changes using `git commit`.
11. Push the branch to GitHub using `git push`.
12. Create a **Pull Request** and merge the branch with the **main branch**.

## Use Cases

### Use Case 1 – Welcome Message

The program displays a welcome message when the application starts.

Example Output:

```
Welcome to Palindrome Checker App
```

---

### Use Case 2 – Hardcoded Palindrome Check

The program checks whether a **hardcoded string** is a palindrome.

Example:

```
Input: madam
Output: Palindrome
```

---

### Use Case 3 – User Input Palindrome Check

The program allows the user to **enter a string from the keyboard** and checks whether it is a palindrome or not.

Example:

```
Enter a word: level
Output: Palindrome
```
